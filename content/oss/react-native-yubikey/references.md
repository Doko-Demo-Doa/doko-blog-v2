## Glossary

### Terms
- **FIDO2** - Fast Identity Online 2, passwordless authentication standard
- **WebAuthn** - Web Authentication API, browser implementation of FIDO2
- **CTAP** - Client to Authenticator Protocol, defines communication
- **OTP** - One-Time Password, time-based or event-based codes
- **TOTP** - Time-based OTP (RFC 6238)
- **HOTP** - HMAC-based OTP (RFC 4226)
- **MFi** - Made for iPhone, Apple accessory certification
- **NFC** - Near Field Communication, wireless standard
- **USB OTG** - USB On-The-Go, allows devices to act as USB hosts
- **PIV** - Personal Identity Verification, certificate-based auth
- **OpenPGP** - Encryption and signing protocol
- **Attestation** - Proof that credential was created by authentic device
- **Assertion** - Proof of authenticator possession
- **Resident Key** - Credential stored on authenticator device
- **User Verification** - Biometric or PIN verification on device
